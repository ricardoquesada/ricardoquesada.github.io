Commando 2084 - a game for the Commodore 64

Commando 2084 is the mix between Commando and Robotron 2084: it is like the original Commando game, but using the Robotron 2084 controls. You play it using the two joysticks at the same time: Joy #2 controls the hero direction Joy #1 controls the bullets direction But better if you play it with a Unijoysticle 2 with a Dualshok4 gamepad (or similar). https://youtu.be/iXdx9jOd0Ho Download Download game: commando-2084.d64 Source code, fully commented: https://gitlab.com/ricardoquesada/c64-commando-2084/
